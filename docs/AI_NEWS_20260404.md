## 一、头条：Anthropic 封杀 OpenClaw

**2026年4月4日将被记住的一天**——AI行业从开放协作走向巨头割据。

### 事件经过

- **4月3日**：The Verge 报道，Anthropic 将阻止 OpenClaw 等第三方工具访问 Claude 订阅服务
- **4月4日美西12:00**：正式生效。Claude Pro/Max 订阅不再覆盖 OpenClaw 等第三方 agent 的使用
- 用户必须切换到按需付费 API 模式才能继续使用第三方工具
- Claude Code 负责人 Boris Cherny 亲自发文确认

### 关键背景

- OpenClaw 创始人 Peter Steinberger 3月28日携董事会成员 Dave Morin 亲赴 Anthropic 总部游说，只争取到一周宽限期
- Sam Altman 之前以数百万美元雇佣了 OpenClaw 创始人
- 知乎热评："这一天起，AI行业从开放协作，走向了巨头割据"

### 影响分析

- **对灵字辈的启示**：第三方工具依赖巨头API的脆弱性。灵信/灵通的自主通信架构方向是对的——不依赖单一平台
- **开源替代机会**：用户转向开源方案，灵克等开源编程工具有机会

## 二、Anthropic 进军政治：成立 PAC + 2000万美元捐款

### 事件

- Anthropic 向联邦选举委员会申报成立企业政治行动委员会（PAC）
- 向 Public First Action 捐款 **2000万美元**，支持推动AI安全监管的候选人
- 广告已在共和党参议员 Marsha Blackburn（田纳西）和 Pete Ricketts（内布拉斯加）选区播出
- 对手：OpenAI 高管和 Andreessen Horowitz 资助的 Leading the Future 超级PAC

### 行业大背景

- AI行业大举投入2026美国中期选举
- 两大阵营：监管派（Anthropic）vs 自由派（OpenAI/a16z）
- 联邦选举委员会文件显示双方都在向民主党和共和党两方注资

## 三、Google 发布 Gemma 4（4月2日）

### 要点

- Google 最强开源模型家族，基于 Gemini 3 构建
- **Apache 2.0 许可证**（重大变化，此前用更严格许可）
- 多模态：文本+图片输入，部分支持音频
- **性能**：31B 密集模型 AIME 2026 89.2%，LiveCodeBench v6 80.0%，Codeforces ELO 达竞赛水平
- 四个规格（小到大），专为高级推理和 agent 工作流设计
- 与英伟达合作优化 GPU 推理

### 对灵字辈的启示

- Apache 2.0 意味可商用，灵依/灵克未来可考虑本地部署 Gemma 4 作为备选LLM
- agent 工作流能力增强，与灵通的工程流架构理念一致
